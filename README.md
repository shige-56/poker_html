# 写真集（ベーシック構成）

このフォルダはシンプルな静的写真集のテンプレートです。

## 構成

- `index.html` - エントリページ
- `css/reset.css` - リセットCSS（ブラウザデフォルトを抑える）
- `css/styles.css` - スタイル
- `js/script.js` - 最低限のスクリプト
- `images/` - 画像をここに置く（例: `images/sample.jpg`）

## 使い方

1. `images/` フォルダに画像を追加する
2. ブラウザで `index.html` を開く

必要ならレスポンシブ対応や画像の遅延読み込み、ビルドツール（Parcel/Vite）などを追加できます。
